# PorQuem

Sistema de gestão das campanhas de capacitação de recursos para entidades filantrópicas.
 
## Technologies

- React
- React Hook Form
- Redux
- HTML Parser
- React Quill
- React Transition Group
- React Typist
- SweetAlert2
- SCSS
- Axios
- JWT Decode
